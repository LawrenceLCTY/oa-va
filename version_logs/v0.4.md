# v0.4

## Summary

Hybrid voice pipeline for more natural calls without requiring all models to run locally.

## Changes

- Added OpenAI-backed structured answer extraction for patient replies.
- Added OpenAI-backed friendly assistant prompt rewriting.
- Switched `/api/tts` to OpenAI TTS by default.
- Added `/api/stt` for local SenseVoiceSmall transcription.
- Updated browser microphone flow to record audio and send it to the local STT endpoint when available.
- Removed mandatory Kokoro preload from app startup.
- Changed the default opening to a natural demo flow that skips brittle hearing/time/permission gates. Set `STRICT_READINESS_FLOW=1` to restore the older gated protocol flow.

## Safety

- Deterministic clinical state progression remains local.
- Rule-based red-flag detection still runs before and alongside LLM extraction.
- If OpenAI or SenseVoice is unavailable, typed input and rule-based fallback still work.

## Configuration

- `OPENAI_API_KEY`
- `OPENAI_LLM_MODEL`
- `OPENAI_TTS_MODEL`
- `OPENAI_TTS_VOICE`
- `SENSEVOICE_MODEL`
- `SENSEVOICE_DEVICE`
