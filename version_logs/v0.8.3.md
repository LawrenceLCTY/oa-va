# v0.8.3

Date: 2026-07-09

## Goal

Remove the extra demo/trust strip language that was making the interface harder to understand for stakeholders and target users.

- Kept the v0.8 clinical assistant layout, call surface, transcript, progress rail, and doctor report preview intact.
