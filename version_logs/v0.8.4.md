# v0.8.4

Date: 2026-07-09

## Goal

Adapt the v0.8 visual system so it better balances a calm clinical experience for older OA patients with a recognizable Peking University Medical School association.

## Design References

- Peking University official visual identity pages include an identification system and standard color guidance.
- Peking University official navigation links to Peking University Health Science Center.
- The UI uses the requested PKU red value `#94070A` as the primary institutional anchor.

## User-Facing Changes

- Added the official Peking University Health Science Center SVG logo from `bjmu.edu.cn/images/logo.svg` to the top-left brand area.
- Added explicit CSS design tokens for PKU red, a darker PKU red, soft red surfaces, muted medical blue, and care green.
- Recolored the brand mark, primary action, progress completion, report emphasis, and main call surface around PKU red.
- Preserved restrained medical blue for voice/input states so interactive feedback remains distinct from institutional branding and urgent safety states.
- Removed the old teal/purple assistant palette from the main visual system.

## Backend Changes

- None.

## Safety/Clinical-Rule Changes

- None. Urgent and danger states retain separate red treatment and explicit labels.
